# User Story: AI Use Case Inventory

**Story:** As a Chief AI Officer,  
I need a centralized AI Use Case Inventory,  
so that I can maintain visibility and control over all material AI systems in production.

**Acceptance Criteria (Gherkin Syntax):**

**Scenario:** Formal approval of high-cost AI tool requests
Given a new AI tool request costs more than the designated threshold per year,
When the AI Governance Committee conducts a majority vote,
Then the request is formally approved or denied and recorded in the audit trail.
